# My Portfolio and Blog Website

Built with Hugo, JavaScript, and TailwindCSS

## Running Hugo on local connection

```bash
hugo server --bind 0.0.0.0 --baseURL http://<YOUR_IP> --port 1313 --disableFastRender
```
